# SocialStoryScript - Teacher Demonstration
# Demonstrates ALL lexical features with proper context

Go live.

# ===== 1. ACCOUNT CREATION (With names!) =====
The account TechInfluencer was created.
The account FashionBlogger was created.
The account GamingChannel was created.

# ===== 2. VARIABLE INITIALIZATION (Account-specific) =====
The account TechInfluencer started with 1000 followers.
The account TechInfluencer started with 500 likes.
The account TechInfluencer began at 3.5 engagement rate.
The account TechInfluencer now has 2500 reach.

The account FashionBlogger started with 2000 followers.
The account FashionBlogger started with 1500 likes.
The account FashionBlogger began at 4.2 engagement rate.

# ===== 3. STORY POSTING (Account-specific) =====
The account TechInfluencer posted a story about "AI Revolution" that had 5000 views.
The account FashionBlogger posted a story about "Summer Trends" that had 3000 views.

# ===== 4. ACTION VERBS (Account-specific operations) =====
The account TechInfluencer gained 50 likes.
The account TechInfluencer lost 10 followers.
The account FashionBlogger gained 100 followers.
The account FashionBlogger added another story "Fashion Tips"

# ===== 5. INCREMENT/DECREMENT (Account-specific) =====
The account TechInfluencer increased by 1.5 engagement rate.
The account TechInfluencer decreased by 0.2 growth rate.
The account FashionBlogger increased by 0.3 engagement rate.

# ===== 6. ARITHMETIC EXPRESSIONS (With context) =====
The account TechInfluencer now has likes plus followers reach.
The account FashionBlogger now has views minus comments result.
The account GamingChannel now has posts times 3 total.

# ===== 7. COMPLEX EXPRESSIONS =====
The account TechInfluencer now has (likes plus followers) times 2 reach.

# ===== 8. CONDITIONALS (Account-specific) =====
When TechInfluencer likes more than 100
{
    Announce "TechInfluencer is growing"
    The account TechInfluencer gained 50 followers.
}

When FashionBlogger followers less than 5000
{
    Display "FashionBlogger needs more followers"
}

When TechInfluencer views exactly 5000
{
    Announce "Perfect milestone for TechInfluencer"
}

When FashionBlogger engagement rate at least 3.0
{
    Display "FashionBlogger has good engagement"
}

When TechInfluencer likes exceeded 1000
{
    Announce "TechInfluencer went viral"
}

# ===== 9. LOGICAL OPERATORS (With context) =====
When TechInfluencer likes more than 100 also TechInfluencer followers more than 50
{
    Announce "TechInfluencer meets both criteria"
}

When FashionBlogger views exceeded 500 either FashionBlogger comments more than 20
{
    Display "FashionBlogger has engagement"
}

# ===== 10. LOOPS - Time-based (Account-specific) =====
Every day for 7 days,
{
    The account TechInfluencer gained 10 likes.
    The account FashionBlogger gained 15 likes.
}

# ===== 11. LOOPS - Variable increment (Account-specific) =====
Every 2 days incrementing by 2
{
    The account TechInfluencer gained 20 followers.
}

# ===== 12. LOOPS - Range (Account-specific) =====
For each follower from 1 to 100
{
    The account TechInfluencer gained 1 views.
}

# ===== 13. LOOPS - Collection (Account-specific) =====
For each post in TechInfluencer feed
{
    The post gained 5 likes.
}

For each story in FashionBlogger feed
{
    The story gained 10 views.
}

# ===== 14. LOOP CONTROL (Account-specific) =====
For each follower from 1 to 100
{
    When TechInfluencer followers exceeded 50
    {
        Stop the story.
    }
    The account TechInfluencer gained 1 views.
}

# ===== 15. NESTED STRUCTURES (Account-specific) =====
When TechInfluencer likes exceeded 100
{
    Announce "Level 1: TechInfluencer popular"
    When TechInfluencer followers more than 500
    {
        Display "Level 2: TechInfluencer influential"
        When TechInfluencer engagement rate at least 3.0
        {
            Announce "Level 3: TechInfluencer is superstar"
        }
    }
}

# ===== 16. FUNCTION DEFINITION =====
The story of calculateTotal begins with accountName, metric1, metric2
{
    The account now has metric1 plus metric2 result.
    Tell back result.
}
The story ends.

The story of getEngagement begins with accountName, likes, views
{
    The account now has likes divided evenly by views rate.
    Tell back rate.
}
The story ends.

# ===== 17. FUNCTION CALLS (With parameters) =====
Tell calculateTotal(TechInfluencer, 100, 200).
Tell getEngagement(FashionBlogger, 1000, 5000).

# ===== 18. BUILT-IN FUNCTIONS (With parameters) =====
Calculate virality(TechInfluencer).
Calculate engagement with TechInfluencer likes, TechInfluencer views.
Find top post(TechInfluencer).
Find the highest reach in TechInfluencer feed.
Find the total reach in FashionBlogger feed.
Count total engagement(TechInfluencer).
Check if trending(FashionBlogger).
Analyze growth(TechInfluencer).

# ===== 19. ARRAY OPERATIONS (Account-specific) =====
Find max viral account(TechInfluencer, FashionBlogger, GamingChannel).

# ===== 20. STRING OPERATIONS (Account-specific) =====
Reverse the caption(TechInfluencer).
Detect spam in TechInfluencer caption.
Clamp engagement between 0.0 and 5.0 for TechInfluencer.
Reverse growth(TechInfluencer).
