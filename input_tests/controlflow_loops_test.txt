Go live.
Announce "controlflow loops".

// --- BASELINE ACCOUNT ---
The account Flow was created.
The account Flow started with 0 likes.
The account Flow started with 10 followers.
The account Flow started with 0 comments.

// --- 1) EVERY-DAY LOOP + CONTINUE (Skip this post) ---
Every day for 3 days,
{
    The account Flow gained 2 likes.
    When Flow likes more than 2
    {
        Skip this post.
    }
    The account Flow gained 100 likes.
}

// --- 2) INCREMENTING LOOP ---
Every 4 days incrementing by 2
{
    The account Flow gained 1 followers.
}

// --- 3) DECREMENTING LOOP ---
Every 4 days decrementing by 2
{
    The account Flow gained 1 comments.
}

// --- 4) RANGE LOOP ---
For each follower from 1 to 3
{
    The account Flow gained 1 shares.
}

// --- 5) TRENDING LOOP + BREAK (Stop the story) ---
trending loop until Flow likes more than 20
{
    The account Flow gained 5 likes.
    When Flow likes more than 15 also Flow followers less than 5 either Flow comments more than 0
    {
        Stop the story.
    }
}

// --- 6) IF / OTHERWISE IF / OTHERWISE CHAIN ---
When Flow likes more than 20
{
    Announce "if branch".
}
Otherwise if Flow likes exceeded 10
{
    Announce "elseif branch".
}
Otherwise
{
    Announce "else branch".
}

// --- 7) OPPOSITE (NOT) ---
When opposite Flow followers more than 9999
{
    Announce "opposite branch taken".
}

// --- 8) AND (also) ---
When Flow likes more than 5 also Flow followers more than 0
{
    Announce "and branch taken".
}

// --- 9) OR (either) ---
When Flow likes more than 9999 either Flow followers more than 0
{
    Announce "or branch taken".
}

Display Flow likes.
End live.
