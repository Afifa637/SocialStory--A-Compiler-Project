# SocialStoryScript Demo Programs

These demos focus on practical social-media analytics workflows.

1. `01_account_baseline_report.txt` — Baseline account setup and quick health report.
2. `02_engagement_health_check.txt` — Rule-based engagement classification.
3. `03_30_day_growth_simulation.txt` — Loop-driven monthly growth simulation.
4. `04_campaign_a_b_comparison.txt` — Campaign comparison and winner decision.
5. `05_input_driven_strategy.txt` — Input-driven strategy recommendation.
6. `06_feed_weekly_post_analysis.txt` — Weekly feed/post analytics workflow.
7. `07_builtins_analytics_report.txt` — Built-in analytics report pipeline.
8. `08_optimizer_meaningful_workload.txt` — Meaningful workload for TAC optimization output.
